# Heart-Disease-Prediction

Project Description

This project predicts whether a person has heart disease using machine learning algorithms based on medical attributes such as age, cholesterol, blood pressure, chest pain type, and heart rate.

The system helps in early diagnosis and healthcare decision-making.
